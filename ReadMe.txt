The code main.py is written in PySpark version 2


export SPARK_MAJOR_VERSION=2

Keep main.py in the main directory (Example in my case, /home/hari9876/hari/dl/de)

How to execute the code - Example 
==================================
spark-submit main.py  /home/hari9876/hari/dl/de

Note the first parameter /home/hari9876/hari/dl/de is the input directory where the program expects the weather*.csv files to be available.
